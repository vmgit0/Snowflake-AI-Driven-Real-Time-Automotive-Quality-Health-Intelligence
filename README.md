An AI-powered data platform built on Snowflake for real-time monitoring, anomaly detection, and predictive maintenance of Electric Vehicles (EVs).

The system processes EV telemetry data (battery, motor, and usage signals) to detect anomalies, analyze root causes, and predict failures up to 30 days in advance.

### 🔑 Key Features

* Real-time anomaly detection (battery, motor, usage)
* 30-day failure forecasting (High/Medium/Low risk)
* AI-driven root cause analysis using Snowflake Cortex
* Explainable health metrics (Battery, Motor, Usage scores)
* Interactive Streamlit dashboard for insights

### ⚙️ Tech Stack

Snowflake • SQL • Snowflake Cortex AI • Python • Streamlit

### 💼 Impact

Enables proactive maintenance, reduces downtime, and improves EV reliability through data-driven decision-making.
